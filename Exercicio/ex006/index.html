Trabalhando com Listas

Listas Ordenadas

Simbolizadas pela tag <ol>, uma lista ordenada é onde os itens estão dispostos em ordem, seja de importancia seja por ordem cronológica, ela tem esse sentido, dentro da tag, abrimos outra tag chamada <li> pata cada item que queremos adicionar, por padrão essa lista vem com números naturais em ordem de adição na frente de cada item, podemos trocar isso se quisermos.

  1. Primeiro colocamos a tag <ol>
  2. Depois abrimos uma tag <li> para cada item que quisermos adicionar
  3. Depois verificamos como a lista ficou
  4. E se quisermos, a editamos

Listas não ordenadas

Simbolizadas pela tag <ul> são listas onde a ordem em que os itens estão não tem importancia, o processo de criação delas é o mesmo, o quie muda é que agora ganhamos uma bolinha antes de cada item, podemos mudar isso também.

  • Pular Corda
  • Andar a cavalo
  • Andar de Bicicleta
  • Nadar
