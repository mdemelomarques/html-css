CSS, Estilo Externo

É o estilo feito numa página separada do documento original, ela é anexada a ele por meio da tag link, que inline recebe o diretório do arquivo css com as alterações, depois colocamos o atributo rel="stylesheet"e assim nossas alterações são feitas por meio dessa CSS externa, esse metodo é o mais usado por permitir a estilização de varias paginas com um unico arquivo, bastando apenas linka-lo aos documentos de página, esse metodo deixa o código ainda mais organizasdo, pois separa a parte de conteudo do codigo cda parte de estilo

O CSS externo funciona da mesma forma que o interno, com seletores e declarações, a diferença é que não precisamos abrir nenhuma tag <style> nele
