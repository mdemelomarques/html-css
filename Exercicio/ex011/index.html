Inserindo Vídeos Hospedados Localmente

Adicionar videos armazenados localmente é bem fácil, com a tag <video> colocamos uma tag <source> com o caminho do video, é bom ter varias sources para um único video, pra aumentrar a gama de compatilidade, e ter nosso conteúdo exibido em qualquer navegador, assim como em audios, precisamos do atributo controls para criar a aŕea de interação para o usuário, podemos adicionar uma thumb para o video com o atributo poster, tudo inline na tag video

Inserindo videos hospedados externamente

Basta procurar plataformas que tenham essa opçãos, a mais famosa é o youtube, mas temos também vimeo e google drive, geralmente basta clicar em compartilhar e depois em incorporar, eles nos dão o código de incorporação pra jogarmos direto no nosso código html
