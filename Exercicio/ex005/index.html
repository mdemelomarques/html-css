Principais Formatações

Negrito/Destaque

Nesse paragráfo usaremos duas formas de dar destaque, a tag <b> e a tag <strong>, quando eu dou destaquea uma palavra usando <b>, eu estou meramente mudando sua forma, mas o meu navegador não sabe que quero dar destaque aquele termo, é apenas uma mudança de visual, para não só mudar o visual, mas também dar significado para aquele texto, precisamos usar a tag <strong>, assim,,esse texto não fica só visualmente destacado, mas semanticamente também

Itálico/ênfase

A mesma coisa acontece com o itálico, podemps deixar um texto visualmente em itálico usando a tag <i> mas isso moeramente muda sua aparência:,Veja minha aparẽncia, mas o navegador não sabe que estou dando enfâse pra essa palavra, pra dar significado, usamos a tag <em>, e ela ganha não só forma, mas significado.

Tag <mark>

Podemos passar um marca texto sobre um texto, com a tag mark, por padrão ele fica Amarelo, porém, com CSS, podemos mudar sua cor para, por exemplo verde.

Tag <small>

Com essa tag deixamos nosso texto bem pequenininho, ela é semântica também

Tag <big>

Era usada pra deixar um texto levemente maior hoje em dia está obsoleta, embora ainda funcione.

Tag <del>

Diz para o navegador que aquela palavra ali escrita, apesar de visivel, foi deletada e não deve ser considerada

Tag <ins>

Essa tag sublinha a palavra, seu significado é que aquele texto foi inserido depois do texto original, é uma nova adição, a tag <u> tem a mesma forma, mas não tem o significado semântico da tag <ins>

Tag <sup> e <sub>

com a tag sub, nosso texto abrangido por ela fica acima da palavra, como numa equação matemática, por exemplo: X+24+3-22, já a tag sub, faz com que o texto englobado fique abaixo, por exemplo o 2 em H2O

Tag <pre>

Essa tag faz com que o texto apareça pré formatado, ou seja, da exata maneira que escrevemos no código, não importa como escrevamos, isso é bom pra blocos de código ou poesias 

        Rosas são vermelhas
        Violetas são azuis
        Essa foi a tag pre
        Não sei rimar, não é minha praia

Tag <abbr>

É uma tag feita pra abreviações, colocando um titulo sobre elas quando o mouse do usuário passa por cima da palavra, é importante pra semantica do conteúdo e pra acessibilidade pea leitores de tela, colocamos a tag em volta da palavra, e inline atribuimos um titulo pra ela, pra exemplificar, veja o significado da abreviação WWW

Tags <bdo>

Essa tag é usada pra sobesvrever a direção de leitura de uma palavra ou texto, precisamos definir a direção de leitura inline nessa tag, dir=rtl pra colocarmos da direita pra esquerda e dir=ltr da esquerda pra direita, pra exemplificar, todo o texto depois de ":" estará invertido: Esse texto está de trás pra frente

Tag <code>

É uma tag que serve pra indicar que o texto ali inserido faz parte de um código de computador, o conteúdo é nostrado de forma monoespaçada, ou seja, todas as letras com o mesmo espaço, exemplo: selectAll(), push().

Tag <q>

Essa tag serve pra citações simples, visualmente ela só adiciona aspas, mas, em html tudo e semantico, tudo tem seu sentido, ao enveloparmos uma palavra com essa tag, dizemos ao navegador exatamente do que se trata: uma citação de um terceiro que agora faz parte do seu texto, para exemplo, uma frase do seriado Chaves: Eu prefiro morrer do que perder a vida.

Tag <blockquote>

Essa tag serve pra citações grandes de texto, visualmente ela aparece separada do texto normal, mas semanticamente ela explica pro navegador que aquilo é uma citação em bloco, usando o atributo cite inline nessa tag, podemos colocar a fonte dessa citação, como um link para site, embora visualmente não tenha efeito nenhum, para o naegador e mecanismos de busca, isso importa, exemplo:, segundo Maria Julia Paes da Silva, em seu livro: Qual o tempo do cuidado?
