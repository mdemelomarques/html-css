Links Internos

Para links internos, não precisamos colocar toda a url, apenas o nome e diretório do documenyo da nossa pagina, como é uma página dentro do nosso próprio site, não precisamos configurar o target se não quisermos, pois poir padrão ela abre na mesma página, mas se quisermos especificar, podemos colocar target="_self", para termos o mesmo efeito e deixar semantico, se o arquivo está na mesma pasta, basta colocarmos o nome em href, se não, colocamos a pasta/nome, colocamos rel=next e rel=previous respeitando a hierarquia das páginas do site, por exemplo: 01 pra 02, rel=next, 02 pra 01, rel=previous, e assim por diante

Para paginas na mesma pasta só precisamos colocar o nome no href=, para páginas em pastas abaixo da nossa, usamos nome da pasta/nome do arquivo no href, se quisermos subir uma pasta, usamos ../nome do arquivo, um ponto para cada repartição a subir, o primeiro ponto representando a própria repartição da página na qual estamos desenvolvendo,adicionamos uma barra a cada dois pontos

Outros tipos de rel=

Temos o rel="nofollow" que diz aos mecanismos de busca pra não atribuir valor ou autoridade pra aquele link, geralmente é usado pra materiais de span e links pagos, pode ser usado pra links comerciais, embora nesses casoso o google recomenda o uso de rel="sponsored"

Links para donwloads

O processo para colocar links para downloads é o mesmo que para outros links internos, exceto que precisamos especificar que queremos baixar essa midia, depois do href do link, abrimos outra propriedade chamada download= e colocamos o nome do arquivo corretamente e completo, tem como fazer esse processo colocando o media type do arquivo, que é a sua extensão, a que vemos depois do ponto no final dele, basta colocar type="media type", para saber o média type de um arquivo basta conferir no site Iana.org
